Resumen del Proyecto

Este sistema detecta imágenes generadas por inteligencia artificial utilizando una red neuronal convolucional (CNN) con MobileNetV2. El objetivo es mitigar la desinformación visual y el uso malicioso de deepfakes.

Metodología

  • Recopilación de datos confiables (dataset Kamblee)
  • Preprocesamiento y aumento de datos
  • Entrenamiento con Transfer Learning (MobileNetV2)
  • Evaluación con métricas de precisión y pérdida
  • Integración en una aplicación web con Flask
